About Archivematica

Archivematica is a suite of open-source tools in an open source wrapper that helps users carry out OAIS-compliant digital preservation tasks on their content.

Archivematica's main function is to create Archival Information Packages (AIPs) for long-term storage.

In the middle /1

We think of Archivematica as part of an ecosystem of digital preservation tools. Just as it fulfils a part of the OAIS requirements, Archivematica fulfils part - the AIP-creating part - of the lifecycle of your content.

In the middle /2

Integrating with repository platforms is a key way that Archivematica continues to grow. Archivematica has integrations (of varying levels of maturity) with Hydra, ArchivesSpace, Islandora, DSpace, Dataverse, CONTENTdm, and others.

In the middle /3

Making sure that integration work emphasises flexibility and generic-ness of both data inputs and outputs (as much as we can) is important for ensuring that growth is both sustainable and useful.

How we develop

Proof-of-concept projects (Hydra, Dataverse) - done as R&D projects, code may not be stable enough to roll into release

Sponsored features/feature enhancements (ArchivesSpace, Islandora) - projects usually include a 10% community fee that pays for merging into next release

Hydra: York/Hull pilot /1

Performed gap analysis to see how Archivematica could meet their research data management needs and identified several areas to improve.

Decided to dedicate project funding to working on:

  • On-demand automated DIP generation
  • Improved METS parsing
  • Introduction of a generic search REST API
  • Ability to handle multiple checksum algorithms
  • Better handling of unidentified files
